Linux/i4lfax/ISDN resources

http://www.linmodems.org/
This is an effort to get vendor support for 'Win-modems' which are basically a soundcard that connects to the phone-line. In MS-Windows, the host CPU takes care of the DSP calculations needed to run fax, modem etc. just like the i4lfax project. The i4lfax project will try to support Win-modems as well as ISDN if possible.

The 'linmodems.org' and 'i4lfax' projects are covering two different aspects of modem technology right now, fax vs. V.34. The two projects have established contact and will try to cooperate where possible.

http://www.pctel.com/
PC-TEL is a company selling Win-Modem solutions, and claims to have working software for Linux.

http://www.mhessler.de/i4lfaq.html
The FAQ for the 'Isdn4Linux' project. This project makes it possible to use some types of ISDN-cards with Linux.

ftp://ftp.aeccom.com/pub/fax4i4l/howto/current/
This site deals with faxing under Linux with an "AVM B1" (active ISDN-card ).
